# pomodori
Pomodori time management app using GTK

# Prerequisite

* gcc
* gtk2
* sqlite3
* cmake

# Build & install

1. clone this repository using git.
2. build using cmake
    1. mkdir Release
    2. cd Release
    3. cmake -DCMAKE_BUILD_TYPE=Release ..
    4. make
    5. now you have the executable "pomodori"!
3. copy .Pomodori to $HOME
4. run pomodori!
